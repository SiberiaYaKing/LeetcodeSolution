### 链表题型总结
* 双指针技巧的应用
    1. 2倍快慢指针（环形链表，链表中间点）
    2. 双链指针 （合并链表，拆分链表）
    3. 长度差双指针 （链表倒数节点）
    4. 滑动窗口 (前后指针需要根据是否满足条件而移动)
* 链表技巧
    1. 虚拟头节点一般在链表有增删操作时优先使用（或者新建链表，一定有增加节点的操作）
    2. 相交列表分头循环遍历，可以找到相交的节点 (相交链表性质)
* 额外技巧
    1. <b>多个元素取最值</b>，如果没有严格的空间限制，<b>优先考虑用堆</b>
    2. 在<b>C++删除链表节点</b>是最容易出现崩溃的，一定要注意<b>顺序</b>：
        * 需要缓存被删节点的后继与前继，
        * 然后删除该节点，
        * 最后将它们的后继与前继连接上
    4. 写<b>循环体的逻辑</b>时需要<b>遵循状态机原则，应用状态机思想</b>：
        * 检查状态是否发生变化，是则进入状态转换逻辑 (检查状态变化)
        * 实现状态转换逻辑，更改各个变量的值 (状态转换逻辑)
        * 最后才更改表示状态的变量 (最后更改状态)
